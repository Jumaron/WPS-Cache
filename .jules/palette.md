## 2024-05-23 - Reusable Toggle Switch Pattern
**Learning:** The `.wpsc-switch` component is a standard checkbox visually styled as a toggle. Adding `role="switch"` is a simple, reusable way to semantically distinguish these from regular checkboxes in this design system.
**Action:** Always add `role="switch"` to inputs wrapped in `.wpsc-switch` to ensure screen readers announce them correctly as toggles.
